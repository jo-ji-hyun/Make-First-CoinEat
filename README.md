# Make-First-CoinEat
## 동전 줍는 소녀
이 프로젝트는 빗물받는 게임에서 영감을 얻어 만든 [동전 줍는 소녀] 입니다.

동전을 먹으면 랜덤한 점수가 상승합니다.

폭탄을 맞으면 Life가 깍이며 0이 되면 게임 오버입니다.

## 기능
자동 이동 (벽에 충돌시 방향 전환)

마우스 왼쪽 클릭 => 방향 전환

마우스 오른쪽 클릭 => 점프

R => 최고 점수 리셋

## 사용된 버전
Unity 2022.3.17f1 (LTS)
